# Escáner de Red Local (LAN)

Un monitor simple desarrollado en Python para identificar dispositivos activos en una red local.

## 🛠️ Características
- Detección automática del segmento de red.
- Escaneo mediante Sockets TCP (compatible con Android/Termux sin root).
- Interfaz de terminal limpia y rápida.

## 🚀 Ejecución
```bash
python escaner.py
